# Coupler.io Skills

Curated skill collection for business data analysis with your favorite AI tools, powered by [Coupler.io](https://coupler.io) MCP server.

## Skills

### Marketing Analytics

Analyze marketing performance data using Coupler.io dataflows. Covers performance reporting, campaign analysis, cross-channel comparisons, and anomaly detection — designed for marketing managers who need actionable insights, not raw data.

**Location:** `marketing/marketing-analytics/`

**Coupler.io MCP tools used:**
- `list-dataflows` — discover available marketing data
- `get-dataflow` — get dataflow details and execution IDs
- `get-schema` — understand data structure and column mappings
- `get-data` — query data with SQL

## Installation

Add this repository as a Claude Code plugin or copy the skill folders into your project's `.claude/skills/` directory.
